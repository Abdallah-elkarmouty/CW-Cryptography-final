How Your Files are Secured

Your files are encrypted using AES-256 encryption before being stored on the server. The AES key used for encryption is itself encrypted using RSA public-key encryption.

AES-256 Encryption

AES (Advanced Encryption Standard) is a symmetric encryption algorithm widely used across the globe to secure data. AES-256 refers to the use of a 256-bit key size, providing a high level of security.

RSA Encryption

RSA is an asymmetric encryption algorithm that uses a pair of keys: a public key for encryption and a private key for decryption. We use RSA to encrypt the AES key, ensuring that only the holder of the private key can decrypt it.

This combination of symmetric and asymmetric encryption ensures both the security and efficiency of file encryption.
